## Trilium 能做什么？

### 1. 树状 + 网状，双重组织方式

笔记可以像文件夹一样层层嵌套，也可以像维基百科一样互相链接。

更厉害的是**克隆功能**：同一篇笔记可以同时出现在多个位置。比如《Python 学习笔记》可以同时放在"编程"和"2024学习计划"两个分类下，修改一处，处处同步。

### 2. 不只是文字，什么都能记

- **富文本编辑**：所见即所得，支持表格、代码块、数学公式
- **Mermaid 图表**：流程图、时序图、甘特图，代码生成
- **思维导图**：直接在笔记里画脑图
- **无限画布**：自由拖拽，像白板一样思考
- **关系地图**：可视化笔记之间的关联
- **代码编辑**：50+ 语言语法高亮，程序员友好

### 3. 网页剪藏，一键收藏

Chrome/Firefox 扩展，看到好文章一键保存到 Trilium。支持保存整个网页、选中内容或截图。

比印象笔记的剪藏还好用——因为没有数量限制。

### 4. 强大的搜索

- 全文搜索：秒级响应
- 模糊搜索：打几个字就能找到
- 高级筛选：按标签、父级、日期、属性组合查询
- 保存搜索：常用搜索条件一键调用

### 5. 数据安全，自己掌控

- **本地存储**：数据就是一个 SQLite 文件，在你的电脑上
- **加密笔记**：敏感内容 AES-128 加密，密码只有你知道
- **版本历史**：自动保存历史版本，误删可恢复
- **完全离线**：没网也能正常使用

### 6. 日历、看板、表格视图

- **日历视图**：按日期查看笔记，像日记本
- **看板视图**：拖拽管理任务，类似 Trello
- **表格视图**：批量管理笔记属性

这些功能在 Notion 要付费，在 Trilium **全部免费**。

## 怎么开始使用？

### 普通用户：下载即用

**Windows / Mac / Linux 用户**：

1. 访问 [GitHub Releases](https://github.com/TriliumNext/Trilium/releases) 或 [官网](https://triliumnotes.org/)
2. 下载对应系统的安装包
3. 安装，打开，开始使用

就这么简单。不需要注册账号，不需要配置服务器，下载就能用。

数据保存在本地，关掉软件数据还在，换电脑把数据文件夹拷走就行。

### 进阶用户：自托管同步

如果你有自己的服务器（VPS），可以部署 Trilium Server 版本：

- **多设备同步**：电脑、手机、平板数据实时同步
- **网页访问**：任何设备打开浏览器就能用
- **手机端**：虽然没有 App，但网页版体验不错，还支持 PWA

部署方式：Docker 一行命令，或者下载 Server 版直接运行。官方文档写得很清楚，有 VPS 基础的半小时搞定。

## 和付费软件比，缺什么？

说了这么多优点，Trilium 有缺点吗？当然有：

| 方面 | Trilium 的不足 | 适合谁 |
|------|---------------|-------|
| **协作** | 单人使用，不支持多人协作 | 个人知识管理 |
| **移动端** | 没有原生 App，网页版凑合 | 主要在电脑上使用 |
| **界面** | 功能优先，颜值一般 | 不追求花哨 |
| **学习曲线** | 功能多，需要时间熟悉 | 愿意花时间研究 |

**如果你需要团队协作**，Notion 更合适。

**如果你主要在手机上记笔记**，其他工具体验更好。

**但如果你是个人用户，主要在电脑上整理知识**，Trilium 几乎是性价比最高的选择。

## AI 时代的隐藏王牌：免费开放的 API

在 ChatGPT、Claude 等 AI 大模型普及的今天，**笔记软件的 API 开放程度**变得前所未有的重要。

为什么？因为 AI + API = 无限可能。

先看看主流软件的 API 情况：

| 软件 | API 开放程度 | 限制 |
|------|-------------|------|
| **Notion** | 有，但有限制 | 免费版数据库限 100 行，AI 功能需 [$20/月](https://www.notion.com/pricing) |
| **印象笔记** | 需申请审批 | [5 个工作日审核](https://dev.evernote.com/doc/)，流程不透明 |
| **Obsidian** | 官方无 API | 需[第三方插件](https://github.com/coddingtonbear/obsidian-local-rest-api)，同步 $8/月 |
| **Trilium** | **完全开放** | **无任何限制，免费使用** |

Trilium 的 [ETAPI](https://docs.triliumnotes.org/user-guide/advanced-usage/etapi) 提供完整的 REST 接口，配合 Python 库 [trilium-py](https://github.com/Nriver/trilium-py)，几行代码就能操作你的知识库。

### 这意味着什么？

**当 AI 遇上开放 API，你的笔记本就变成了智能助手的大脑。**

举几个真实可实现的例子：

- **智能问答**："我之前写过关于 Docker 部署的笔记吗？" → AI 搜索你的知识库，总结相关内容
- **自动摘要**：长文章一键生成摘要，自动打标签分类
- **周报生成**：AI 汇总你本周的笔记，生成工作周报
- **自然语言管理**："提醒我明天下午 3 点开会" → 自动创建待办事项
- **知识关联**：AI 分析笔记内容，自动建立相关笔记之间的链接

这些功能在付费软件里要么不支持，要么需要额外订阅 AI 服务。

**而 Trilium + 你自己的 AI（Claude API / OpenAI）= 完全自主的智能知识库。**

这才是免费开源的真正价值：**不是省了几十块订阅费，而是获得了无限的可能性。**

唯一的限制，是你的想象力。
